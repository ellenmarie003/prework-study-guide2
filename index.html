Prework Study Guide

HTML

  • HTML is the structure of the webpage and is made up of elements that work togther to provide content and framework.
  • We can liken HTML to the frame of the house.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • All elements of a website that we want viewed or interacted with by users have to be placed within the body element.
CSS
  • Cascading style sheets, or CSS, is the language used to describe a website's presentation
  • CSS describes how each element is rendered
Git
  • Git is an open source control system that allows developers to collaborate on projecs without overwriting each other's work.
  • Through Git, you can revert to prior versions of an application if a problem is discovered.
  • git status: checks what branch we are currently in
  • git checkout -b branch-name: creates a new branch and switches to it
JavaScript
  • JavaScript is a lightweight programming language used to provide logic and interactivity to a website.
